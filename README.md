# Web Security Labs

This repository contains intentionally vulnerable web applications created for learning web security.  

The purpose is to understand common web vulnerabilities and practice securing web applications.

## Current Labs

- SQL Injection

*(More labs like XSS, Command Injection, and File Upload Vulnerabilities will be added over time.)*

## Disclaimer

These applications are intentionally vulnerable and created for educational purposes only.  
Do **not** deploy them on production servers.
